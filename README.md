# PredProfLast
Привет! Это выполнение кейса "Управление столовой" сделанное за 4 дня). (к сожалению очень криво) :(
При выполнении кейса использовался фреймворк Flask. 
В папке templates лежат все html файлы. Файлы с js находятся в static/js.
Для запуска проекта у себя(никто кроме тебя не сможет пользоваться) скачай все файлы и запусти файл flsk_app.py - вся логика сервера.
в файле base.py находятся функции для работы с базами данных.

# хостинг
Для того чтобы захостить приложение, я использую pythonanywhere. 
шаги для успешного запуска:
1) как не странно нужно создать аккаунт
2) нажать на добавить или создать сайт
3) в всплывшем окне выбрать версию python желательно последнюю или предпоследнюю(проект написан на 3.13) и фреймворк Flask, изменить название файла в пути на flsk_app(или другое, но тогда придётся переименовывать файл flsk_app.py)
4) перейти в раздел файлы и в директорию mysite создать папку templates и static. В папке static создать папку js и загрузить все файлы js. В папку templates загрузить все html файлы.
5) в файле flsk_app.py(если вы его не переименовывали) убираем: app.run(debug=True) из if __name__ == '__main__': выносим:
    scheduler.init_app(app) # обязательно
    scheduler.start() # обязательно
    base.Start() # обязательно
    base.AddCookAndAdmin("fedr", "12345", "admins") # добавляем администратора
    base.AddCookAndAdmin("ivan", "2345", "cooks") # добавляем повара
    наполняем меню
    base.AddFood("пятница", "завтрак", "1) суп\n 2) макароны\n 3) чай", "800", 10, 400)
    base.AddFood("понедельник", "завтрак", "1) каша\n 2) макароны\n 3) чай", "500", 5, 250)
    base.AddFood("вторник", "обед", "1) суп\n 2) макароны\n 3) чай", "600", 6, 300)
    base.AddFood("понедельник", "обед", "1) мясо\n 2) макароны\n 3) чай", "20000", 2, 1000)
6) в директорию mysite загружаем mydb.sql (она почему-то не создаётся сама)
7) переходим во вкладку web и в разделе static files указываем путь к js файлам home(может само подставиться)/имя вашего аккаунта/mysite/static/js
8) нажимаем на кнопку Reload и переходим на наш сайт
