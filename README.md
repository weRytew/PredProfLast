# PredProfLast
Привет! Это выполнение кейса "Управление столовой" сделанное за 4 дня в одиночку). (к сожалению очень криво) :(
При выполнении кейса использовался Фреймворк Flask. 
В папке templates лежат все html файлы. Файлы с js находятся в static/js.
Для запуска проекта у себя(никто кроме тебя не сможет пользоваться) скачай все файлы и запусти файл app.py - вся логика сервера.
в файле base.py находятся функции для работы с базами данных.
