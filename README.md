# PredProfLast
Привет! Это выполнение кейса "Управление столовой". В нем ты сможешь зарегистрироваться на сайте столовой, просматривать меню, оплачивать товары, оставлять комментарии итд.
При выполнении кейса использовался фреймворк Flask. 
В папке templates лежат все html файлы. Файлы с js находятся в static/js.

# Установка и запуск
1. Скачайте zip всего репозитория и распакуйте его
2. Скачайте Docker если у вас его нет
3. Откройте распакованную папку в своей среде разработки
4. В консоли напишите docker build -t какое-нибудь имя . (точка обязательна)
5. Если установка прошла без ошибок запустите созданный image во вкладке Docker Desktop или через команду docker run -d -p 3000:5000 какое-то имя
6. Откройте браузер и в адресной строке напишите 0.0.0.0:5000
7. !!! Юху все работает !!!
